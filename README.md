# NIT Goa Campus Map

## Overview
This project is a React/Expo app that displays the NIT Goa campus map with academic buildings, hostels, facilities, and sports grounds. Users can search, filter by category, and view location details.

## Features
- Interactive map with markers
- Search by name
- Filter by category (Academic, Hostel, Facilities, Sports, Workspace, Entrance)
- Location details with description, timings, contact, and images
- AR Mode (optional)
- Mobile-friendly drawer and detail panels

## Categories
- **Academic:** Lecture halls, departments, seminar halls  
- **Hostels:** Boys and Girls hostels  
- **Facilities:** Library, Cafeteria, Gym, Workshop, Parking  
- **Sports:** Sport ground, playgrounds  
- **Workspace:** Mechanical and Civil workshops  
- **Entrance:** Main Gate
